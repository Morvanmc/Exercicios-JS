Exercício 02

Crie uma função que receba uma idade em anos
e retorne a mesma idade em dias.
* Desconsiderar anos bissextos.

JavaScript:

                
                    function anoEmDias() {
                        const  age = document.getElementById('age-c2').value;
                        document.getElementById('resultado').innerHTML = `Idade em dias = ${age * 365}`;
                    }
