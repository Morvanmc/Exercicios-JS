Exercício 12

Crie uma função que receba 2 valores:
- O primeiro deve ser um ojeto com propriedade e valores (EX.: {"a": 1, "b": "Olá"})
- O segundo deve ser o nome de prorpiedade do objeto passado (EX.: "a")
- Retorne um novo objeto sem a propriedade passada.
- Para testar se o objeto não é o mesmo utilize a função Object.is()
- O retorno false indica que não é o mesmo objeto.

JavaScript:

                
                    function removeProperty() {
                        const obj = document.getElementById('obj-c12').value;
                        let prop = document.getElementById('prt-c12').value;
                        prop = prop.replace(/"/g,""); //regex para retirar aspas ""
                    
                        const objeto = JSON.parse(obj);
                        let objCopia = Object.assign({}, objeto);
                        delete objCopia[prop]
                    
                        objCopia = JSON.stringify(objCopia)
                    
                        document.getElementById('resultado').innerHTML = `Objeto: ${objCopia} : ${Object.is(objCopia)}`
                    }
