Exercício 27

Crie uma função que receba um Objeto:
- O retorno deve outro Objeto, porém com os parâmetros chave/valor invertidos.
- Ex.: {"nome": "Maria", "idade": 28} // {"Maria": "nome", 28: "idade"}

JavaScript:

                
                    function inverterObj() {
                        const obj = JSON.parse(document.getElementById('obj-c27').value);
                        let result = []
                        for(let key in obj ) {
                            result.push(obj[key], key)
                        }
                    
                        result = JSON.stringify(result)
                    
                        document.getElementById('resultado').innerHTML = `${result}`
                    }
