Exercício 24

Crie uma função que receba 2 parâmetros:
- O primeiro será um caracter.
- O segundo será uma string.
- O retorno deverá ser a quantidade de vezes que o caracter aparece na string.
- A função deverá ser case-sensitive(diferenciar maiúsculas de minúsculas).
-Ex.: "a", "A sorte favorece os audazes" // retornará 3.

JavaScript:

                
                    function contarCaracter() {
                        let crt = document.getElementById('crt-c24').value;
                        crt = crt.replace(/"/g,""); //regex para retirar aspas ""
                        const str = document.getElementById('str-c24').value.split('');
                        
                        let cont = 0
                    
                        for(let i in str) {
                            crt === str[i] ? cont++ : cont
                        }
                    
                        document.getElementById('resultado').innerHTML = `Total de vezes que o caracter aparece é = ${cont}`
                    }
