Exercício 23

Crie uma função que receba uma string:
- Conte o múmero de palavras na string.
- Considere que todas as palavras só são separadas por um único espaço.
- Ex.: "Sou uma frase" // retornará 3.

JavaScript:

                
                    function contarPalavras() {
                        const str = document.getElementById('str-c23').value;
                        const total = str.split(' ').length
                    
                        document.getElementById('resultado').innerHTML = `Total de palavras = ${total}`
                    }
