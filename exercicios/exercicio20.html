Exercício 20

Crie uma função que receba a base e altura de um triângulo:
- Retorne a área do triângulo ((base x altura)/2).
- O resultado deve ter a precisão de duas casas decimais.

JavaScript:

                
                    function calcularArea() {
                        const b = parseFloat(document.getElementById('base-c20').value);
                        const h = parseFloat(document.getElementById('altura-c20').value);
                    
                        const area = (b * h) / 2;
                    
                        document.getElementById('resultado').innerHTML = `Área = ${area.toFixed(2)}`
                    }
