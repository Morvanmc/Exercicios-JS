Exercício 14

Crie uma função que receba 1 Objeto:
- O retorno deve ser um array de arrays onde chave/valor sejam os elementos do array
- Ex.: {"nome": "Maria", "idade": 28} => [["nome", "Maria"], ["idade", 28]]

JavaScript:

                
                    function objToArray() {
                        const obj = JSON.parse(document.getElementById('obj-c14').value);
                        let result = []
                        for(let key in obj ) {
                            result.push([key, obj[key]])
                        }
                        result = JSON.stringify(result)
                    
                        document.getElementById('resultado').innerHTML = `${result}`
                    }
