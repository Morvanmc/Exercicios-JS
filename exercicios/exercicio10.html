Exercício 10

Crie uma função que receba 1 número:
- O número será a quantidade de vezes que o "+" será repetido.
- O resultado deve ser sem separação(Ex.: "+++++").

JavaScript:

                
                    function repetirPlus() {
                        const num = parseInt(document.getElementById('num-c10').value);
                        result = ''
                        for(let i = 0; i ‹ num; i++) {
                            result += '+'
                        }
                        document.getElementById('resultado').innerHTML = `${result}`
                    }
