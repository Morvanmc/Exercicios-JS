Exercício 09

Crie uma função que receba 2 parâmetros:
- O primeiro será um valor a ser repetido.
- O segundo será o número de vezes que o primeiro será repetido.

JavaScript:

                
                    function repetir() {
                        const param = document.getElementById('param-c9').value;
                        const num = parseInt(document.getElementById('num-c9').value);
                        result = []
                        for(let i = 0; i ‹ num; i++) {
                            result.push(param)
                        }
                        document.getElementById('resultado').innerHTML = `${result}`
                    }
