Exercício 18

Crie uma função que receba um array de produtos:
- Cada produto deve possuir preço.
- Retorne o total dos produtos.
- Ex.: {"nome": "Jornal online", "categoria": "Informação", "preco": 89.99},
{"nome": "Cinema", "categoria": "Entretenimento", "preco": 150}.
- Retornará 239.99 .

JavaScript:

                
                    function somarTotalProd() {
                        const obj = JSON.parse(document.getElementById('array-c18').value);
                        let soma = 0;
                    
                        for(let p of obj) {
                            soma += p.preco;
                        }
                    
                        document.getElementById('resultado').innerHTML = `${soma}`
                    }
