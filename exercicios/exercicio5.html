Exercício 05

Crie uma função que receba:
- Dois números;
O retono deve ser true se o primeiro for >= ao segundo.
O retorno deve ser false caso não satisfaça a primeira condição
e caso os valores sejam de tipos diferentes(Number/String).

JavaScript:

                
                    function trueOrFalse() {
                        const num1 = parseInt(document.getElementById('PrimeiroNum-c5').value);
                        const num2 = parseInt(document.getElementById('SegundoNum-c5').value);
                    
                        if(typeof num1 !== typeof num2) {
                            document.getElementById('resultado').innerHTML = false;
                        } else {
                            document.getElementById('resultado').innerHTML = `${num1 >= num2}`;
                        }
                    }
